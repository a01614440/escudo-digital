# F6.E - WhatsApp / Chat refine

Fecha: 2026-04-23
Commit base auditado: `50ee7e5` (`fix(sharedActivityUi): restore cn import`)

## Objetivo

Cerrar la experiencia de WhatsApp/chat encima de la stage-dominance de F6.D, pero sin tocar contracts, scoring, renderer ni registry.

## Cambios realizados

- El hilo del chat ahora se declara como `role="log"` con `aria-live="polite"` y `aria-relevant="additions text"`.
- El compositor gana contexto propio con `sd-chat-composer-help` y `aria-describedby`.
- Desktop deja de sentirse como una sola columna y pasa a una relación real entre hilo e inspector.
- `sd-chat-insight` queda sticky para que el apoyo visual no compita con el hilo.
- En mobile, el compositor gana presencia visual con sticky positioning.
- El bloque legacy de WhatsApp sigue fuera del flujo activo y no se reactivó durante esta subfase.
