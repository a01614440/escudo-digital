# F6.H - WebLab / web simulation refine

Fecha: 2026-04-23

## Objetivo

Dar a WebLab una presencia mas browser-like, mas inmersiva y mas diferencial por tema, sin tocar el renderer, el registry ni el dominio.

## Cambios realizados

- `frontend/src/components/activities/immersive/WebLabActivity.jsx`
  - agrega un tema visual derivado del contenido del sitio;
  - aplica `web-lab-immersive` con `theme-*` para activar la skin existente;
  - introduce browser chrome real con `web-lab-browser-bar`, `web-lab-browser-meta`, `web-lab-browser-tabs` y `web-lab-domain`;
  - convierte la seccion de briefing en `web-lab-mission` y el panel principal en `web-lab-brief`;
  - ensancha el workbench para que el contenido domine mejor la pantalla;
  - expone `data-sd-web-lab-theme` y `data-sd-web-lab-stage` para trazabilidad.
- `test/f6-weblab-refine-smoke.test.js`
  - protege la presencia del browser shell, del tema y de las clases clave.

## Validacion

- `npm-local.cmd test`
- `npm-local.cmd run build`
- `npm-local.cmd run build-storybook`
- `git diff --check`

## Fuera de alcance

No se toco `ActivityRenderer.jsx`, `activityRegistry.js`, hooks de dominio, backend, services, contracts, scoring, `CoursesView.jsx`, `LessonView.jsx`, otras simulaciones ni `legacy.css`.

## Estado

F6.H queda cerrada como refinement de WebLab / web simulation.
